⚡ ¿Qué es JavaScript?

JavaScript (JS) es un lenguaje de programación interpretado que permite crear contenido interactivo. Se ejecuta en el navegador y también en servidores (Node.js).

📦 Variables

Se declaran con let, const o var (desaconsejado).

let nombre = "Ana"; const PI = 3.1416; var antigua = "evitar";

🔢 Tipos de datos

String, Number, Boolean, Array, Object, null, undefined.

let texto = "Hola"; let numero = 42; let activo = true; let lista = [1, 2, 3]; let persona = { nombre: "Luis", edad: 30 };

🛠️ Funciones
