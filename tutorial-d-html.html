📄 HTML HyperText Markup Language

El esqueleto de toda página web. Aprende estructura, etiquetas y buenas prácticas — adaptable hasta 320px.

🧱 ¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) define la estructura y el contenido de una página web mediante etiquetas. No es un lenguaje de programación, sino de marcado.

📌 Ejemplo: un párrafo se escribe con <p>Hola mundo</p>.

<h1>...</h1> para títulos principales.

🏗️ Estructura básica

Todo documento HTML comienza con una declaración y las etiquetas <html>, <head> y <body>.

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <title>Mi página</title> </head> <body> <h1>Título principal</h1> <p>Contenido...</p> </body> </html>

🏷️ Etiquetas de texto

h1 Título

h2 Subtítulo

Negrita, cursiva, subrayado, <code>.

Salto de línea: <br> (ejemplo abajo).

<h1>...</h1> <h2>...</h2> <p>...</p> <strong>negrita</strong> <em>énfasis</em> <br> (salto de línea)

📌 HTML semántico

Usa etiquetas con significado: <header>, <nav>, <main>, <section>, <article>, <footer>.

✅ Mejora la accesibilidad, el SEO y hace el código más mantenible.

Ejemplo de estructura semántica:

<body> <header>...</header> <nav>...</nav> <main> <article>...</article> </main> <footer>...</footer> </body>
