🧠 Lógica de Programación esencial

Pensamiento estructurado, algoritmos y fundamentos para programar en cualquier lenguaje.

🤔 ¿Qué es la lógica de programación?

Es la capacidad de estructurar instrucciones para resolver problemas de forma clara y ordenada. No depende de un lenguaje específico: es la base del pensamiento computacional.

🔷 Entrada → ⚙️ Proceso → 🔶 Salida

Ejemplo: “Sumar dos números” → Entrada (5, 3) → Proceso (suma) → Salida (8).

Una buena lógica evita errores y hace que el código sea fácil de mantener.

📐 Algoritmos

Un algoritmo es una secuencia de pasos finitos para resolver un problema. Se puede representar con pseudocódigo o diagramas de flujo.

// Pseudocódigo: preparar un café INICIO 1. Calentar agua 2. Poner café en filtro 3. Verter agua caliente 4. Servir en taza FIN
💡 Características: preciso, definido, finito. La computadora ejecuta instrucciones paso a paso.

📦 Variables y tipos de datos

Una variable es un espacio en memoria que guarda un valor. Tiene un nombre y un tipo.

🔹 Tipos básicos

numero = 42 texto = "Hola" booleano = true lista = [1,2,3]
// JavaScript (ejemplo) let edad = 25; // número let nombre = "Ana"; // string let esMayor = true; // boolean let colores = ["rojo","azul"];

📌 En pseudocódigo: EDAD ← 25 o nombre = "Ana".

⚖️ Estructuras condicionales

Permiten tomar decisiones según se cumpla o no una condición.

🔸 Si (if) - sino (else)

SI (edad >= 18) ENTONCES ESCRIBIR "Eres mayor de edad" SINO ESCRIBIR "Eres menor de edad" FIN SI

🔎 Prueba interactiva: Ingresa una edad (0-100)

¿Mayor?

📌 if, else if, else. También existe switch para múltiples casos.

🔄 Bucles / ciclos

Repiten un bloque de código mientras se cumpla una condición.

🔁 Mientras (while)

contador = 0 MIENTRAS contador < 5 HACER ESCRIBIR "Vuelta:", contador contador = contador + 1 FIN MIENTRAS

🔂 Para (for)

PARA i DESDE 1 HASTA 5 HACER ESCRIBIR "Número:", i FIN PARA

🛠️ Funciones

Bloques de código reutilizables que realizan una tarea específica. Pueden recibir parámetros y devolver un valor.

FUNCIÓN sumar(a, b) resultado = a + b RETORNAR resultado FIN FUNCIÓN total = sumar(8, 2) // total = 10

💬 Las funciones evitan repetir código y organizan mejor el programa.
