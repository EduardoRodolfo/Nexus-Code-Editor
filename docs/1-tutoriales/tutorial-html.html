Introducción a HTML5

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web mediante etiquetas (tags).

¿Sabías que?

HTML5 es la quinta y actual versión principal del estándar HTML. Se publicó oficialmente en octubre de 2014, pero ya era ampliamente utilizado antes de su publicación formal.

¿Qué es HTML?

HTML no es un lenguaje de programación, sino un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que encierran diferentes partes del contenido para que se vean o comporten de una determinada manera.

Ventajas de HTML5

  • Soporte nativo para audio y video
  • Elementos semánticos mejorados
  • API de almacenamiento local
  • Mejor soporte para aplicaciones web
  • Compatibilidad con dispositivos móviles
Ejemplo Básico de HTML 
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p>Esta es mi primera página web.</p>
</body>
</html>

Estructura Básica de un Documento HTML

Todos los documentos HTML siguen una estructura básica que incluye las siguientes partes esenciales:

Estructura de un Documento HTML
<!DOCTYPE html> - Declaración del tipo de documento
<html> - Elemento raíz
<head> - Información del documento (metadatos)
<body> - Contenido visible de la página
Elemento Descripción Ejemplo
<!DOCTYPE html> Define que el documento es HTML5 Obligatorio al inicio del documento
<html> Elemento raíz que contiene todo el documento <html lang="es">...</html>
<head> Contiene metadatos, enlaces a CSS, título <head><title>Mi Página</title></head>
<body> Contiene el contenido visible de la página <body><h1>Hola</h1></body>
Estructura HTML Completa 
<!-- Declaración del tipo de documento -->
<!DOCTYPE html>

<!-- Elemento raíz con especificación de idioma -->
<html lang="es">

<!-- Cabecera con metadatos -->
<head>
    <!-- Codificación de caracteres -->
    <meta charset="UTF-8">
    
    <!-- Compatibilidad con Internet Explorer -->
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    
    <!-- Viewport para responsive design -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- Título de la página -->
    <title>Mi Página Web</title>
    
    <!-- Enlace a hoja de estilos -->
    <link rel="stylesheet" href="estilos.css">
</head>

<!-- Cuerpo de la página -->
<body>
    <!-- Contenido visible de la página -->
    <h1>Bienvenido a mi página web</h1>
    <p>Este es el contenido principal.</p>
    
    <!-- Enlace a script JavaScript -->
    <script src="script.js"></script>
</body>

</html>

Etiquetas HTML Principales

HTML tiene una gran variedad de etiquetas para diferentes propósitos. A continuación, las más importantes:

Etiquetas de Texto

Formateo de Texto 
<!-- Títulos -->
<h1>Título Principal (solo uno por página)</h1>
<h2>Subtítulo</h2>
<h3>Sub-subtítulo</h3>

<!-- Párrafos -->
<p>Este es un párrafo normal.</p>
<p>Este es otro párrafo con <strong>texto en negrita</strong> 
y <em>texto en cursiva</em>.</p>

<!-- Listas -->
<ul>
    <li>Elemento de lista no ordenada</li>
    <li>Otro elemento</li>
</ul>

<ol>
    <li>Primer elemento ordenado</li>
    <li>Segundo elemento</li>
</ol>

Etiquetas de Enlace e Imagen

Enlaces e Imágenes

Código para Enlaces e Imágenes 
<!-- Enlace (hipervínculo) -->
<a href="https://www.ejemplo.com" target="_blank">
    Texto del enlace
</a>

<!-- Imagen -->
<img src="imagen.jpg" 
     alt="Descripción de la imagen"
     width="300"
     height="200">

<!-- Enlace con imagen -->
<a href="pagina.html">
    <img src="miniatura.jpg" alt="Miniatura">
</a>
