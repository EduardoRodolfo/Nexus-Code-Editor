Introducción a CSS3

CSS (Cascading Style Sheets) es el lenguaje de estilos utilizado para describir la presentación de documentos HTML. Controla el diseño, colores, fuentes y otros aspectos visuales de una página web.

¿Sabías que?

CSS3 es la última versión de CSS e introduce nuevas características como bordes redondeados, sombras, gradientes, transiciones, animaciones y layout más avanzados con Flexbox y Grid.

¿Cómo funciona CSS?

CSS funciona mediante un sistema de reglas. Cada regla consta de un selector y un bloque de declaraciones que contienen propiedades y valores:

Estructura de una Regla CSS 
selector {
    propiedad: valor;
    otra-propiedad: otro-valor;
}

/* Ejemplo concreto */
h1 {
    color: blue;
    font-size: 2rem;
    text-align: center;
}

Formas de Aplicar CSS

CSS Interno
Se define dentro de la etiqueta <style> en el head del documento HTML.
<style>
  body { background: #f0f0f0; }
</style>
CSS Externo
Se escribe en un archivo separado (.css) y se enlaza con <link> en el HTML.
<link rel="stylesheet" href="estilos.css">
CSS en Línea
Se aplica directamente a un elemento HTML usando el atributo style.
<p style="color: red;">Texto rojo</p>

Selectores CSS

Los selectores definen a qué elementos HTML se aplicarán los estilos. CSS tiene una amplia variedad de selectores que permiten un control preciso.

Código CSS 
p {
    color: #264de4;
    font-weight: bold;
}

Tipos de Selectores Comunes

Selector Ejemplo Descripción
elemento p { color: blue; } Selecciona todos los elementos <p>
.clase .destacado { background: yellow; } Selecciona elementos con class="destacado"
#id #header { height: 100px; } Selecciona el elemento con id="header"
[atributo] [target] { border: 1px solid; } Selecciona elementos con el atributo especificado
:pseudo-clase a:hover { color: red; } Selecciona elementos en un estado específico

Box Model de CSS

El Box Model es un concepto fundamental en CSS. Todos los elementos HTML son tratados como "cajas" que constan de:

CONTENIDO
200x150px
padding: 20px
border: 10px
margin: 30px
Contenido (Content)
El área donde se muestra el contenido (texto, imágenes, etc.). Su tamaño se define con width y height.
width: 200px;
height: 150px;
Relleno (Padding)
Espacio entre el contenido y el borde. Es transparente.
padding: 20px;
padding-top: 10px;
Borde (Border)
Bordes que rodean el padding (y el contenido). Puede tener color, estilo y grosor.
border: 2px solid #000;
border-radius: 10px;
Margen (Margin)
Espacio fuera del borde, que separa el elemento de otros elementos.
margin: 30px;
margin-bottom: 20px;
Importante

Por defecto, el width y height especificados en CSS solo incluyen el contenido. Si agregas padding y border, el tamaño total del elemento será mayor. Puedes cambiar esto con box-sizing: border-box para que width y height incluyan padding y border.
