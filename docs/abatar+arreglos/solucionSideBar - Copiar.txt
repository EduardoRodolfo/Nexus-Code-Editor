Funcionalidad de Abrir/Cerrar:

Para ABRIR el sidebar:

· El usuario hace clic en el botón "📁 Explorador" del menú
· El sidebar se expande desde la izquierda
· Todo el contenido se desplaza hacia la derecha

Para CERRAR el sidebar:

· El usuario puede hacer clic en la "×" (equis) del sidebar
· O puede hacer clic nuevamente en el botón "📁 Explorador" del menú
· El sidebar se contrae y el contenido vuelve a su posición original

JavaScript completo para el toggle:

```javascript
// Toggle del sidebar
document.getElementById('toggle-sidebar').addEventListener('click', function(e) {
  e.preventDefault();
  const sidebar = document.getElementById('project-sidebar');
  const container = document.querySelector('.container');
  
  // Alternar estado abierto/cerrado
  const isOpening = !sidebar.classList.contains('open');
  
  sidebar.classList.toggle('open');
  container.classList.toggle('sidebar-open');
  
  // Actualizar el estado visual del botón
  this.classList.toggle('active', isOpening);
});

// Cerrar sidebar con el botón ×
document.getElementById('close-sidebar').addEventListener('click', function() {
  const sidebar = document.getElementById('project-sidebar');
  const container = document.querySelector('.container');
  const toggleBtn = document.getElementById('toggle-sidebar');
  
  sidebar.classList.remove('open');
  container.classList.remove('sidebar-open');
  toggleBtn.classList.remove('active');
});

// Cerrar sidebar al hacer clic fuera (opcional)
document.addEventListener('click', function(e) {
  const sidebar = document.getElementById('project-sidebar');
  const toggleBtn = document.getElementById('toggle-sidebar');
  
  if (sidebar.classList.contains('open') && 
      !sidebar.contains(e.target) && 
      e.target !== toggleBtn && 
      !toggleBtn.contains(e.target)) {
    sidebar.classList.remove('open');
    document.querySelector('.container').classList.remove('sidebar-open');
    toggleBtn.classList.remove('active');
  }
});
```
