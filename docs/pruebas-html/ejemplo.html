Operadores de Asignación en JavaScript

Aprende y prueba los diferentes operadores de asignación con ejemplos interactivos

=
Asignación básica
// Asignación básica let contador = 0; contador = 5; // Asignación simple console.log(contador);
Resultado aparecerá aquí...
// El operador = asigna el valor del lado derecho a la variable del lado izquierdo
+=
Suma y asignación
// Suma y asignación let total = 10; total += 5; // Equivalente a: total = total + 5 console.log(total);
Resultado aparecerá aquí...
// Suma el valor de la derecha a la variable y asigna el resultado
-=
Resta y asignación
// Resta y asignación let saldo = 100; saldo -= 25; // Equivalente a: saldo = saldo - 25 console.log(saldo);
Resultado aparecerá aquí...
// Resta el valor de la derecha a la variable y asigna el resultado
*=
Multiplicación y asignación
// Multiplicación y asignación let precio = 50; precio *= 1.15; // Aumenta el precio en un 15% console.log(precio);
Resultado aparecerá aquí...
// Multiplica la variable por el valor de la derecha y asigna el resultado
/=
División y asignación
// División y asignación let presupuesto = 1000; presupuesto /= 4; // Divide el presupuesto en 4 partes console.log(presupuesto);
Resultado aparecerá aquí...
// Divide la variable entre el valor de la derecha y asigna el resultado
%=
Módulo y asignación
// Módulo y asignación let numero = 17; numero %= 5; // Calcula el residuo de la división console.log(numero);
Resultado aparecerá aquí...
// Calcula el residuo de la división y asigna el resultado
**=
Exponenciación y asignación
// Exponenciación y asignación let base = 3; base **= 4; // Eleva la base a la cuarta potencia console.log(base);
Resultado aparecerá aquí...
// Eleva la variable a la potencia indicada y asigna el resultado
