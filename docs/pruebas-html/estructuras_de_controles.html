🧠 ¿Qué son las estructuras de control?

Las estructuras de control son bloques fundamentales en programación que determinan el flujo de ejecución de tu código. Permiten tomar decisiones y repetir acciones, haciendo que tus programas sean dinámicos y adaptables.

Imagina un semáforo: las estructuras condicionales son como las luces que deciden si los autos deben detenerse o avanzar. Los bucles son como el ciclo repetitivo del semáforo que cambia de rojo a verde una y otra vez.

En JavaScript, existen dos tipos principales:
1️⃣ Condicionales: Deciden qué código ejecutar basado en condiciones
2️⃣ Bucles: Repiten bloques de código mientras se cumpla una condición

🔀 Condicionales: Tomando decisiones

Las estructuras condicionales permiten que tu programa tome decisiones basadas en condiciones. Son como las bifurcaciones en un camino: dependiendo de ciertas condiciones, el programa seguirá un camino u otro.

Buenas prácticas
✅ Usa condiciones claras y descriptivas
✅ Anida condicionales solo cuando sea necesario
✅ Considera usar switch cuando tengas múltiples casos
✅ Evita condiciones demasiado complejas (más de 3 operadores)

if, else if, else

// Verificar la hora del día
let hora = new Date().getHours();
let mensaje;

if (hora < 12) {
    mensaje = "¡Buenos días!";
} else if (hora < 19) {
    mensaje = "¡Buenas tardes!";
} else {
    mensaje = "¡Buenas noches!";
}

console.log(mensaje);

Operador Ternario

// Forma compacta de if/else
let edad = 20;
let puedeConducir = (edad >= 18) ? "Sí, puede conducir" : "No puede conducir";

console.log(puedeConducir);

Switch

// Determinar el día de la semana
let dia = new Date().getDay();
let nombreDia;

switch(dia) {
    case 0: nombreDia = "Domingo"; break;
    case 1: nombreDia = "Lunes"; break;
    case 2: nombreDia = "Martes"; break;
    case 3: nombreDia = "Miércoles"; break;
    case 4: nombreDia = "Jueves"; break;
    case 5: nombreDia = "Viernes"; break;
    case 6: nombreDia = "Sábado"; break;
    default: nombreDia = "Día inválido";
}

console.log(`Hoy es ${nombreDia}`);

🔄 Bucles: Repitiendo acciones

Los bucles permiten ejecutar un bloque de código múltiples veces. Son fundamentales para procesar colecciones de datos y automatizar tareas repetitivas.

Cada tipo de bucle tiene sus casos de uso específicos: - for: Cuando sabes cuántas iteraciones necesitas
- while: Cuando el número de iteraciones es desconocido
- do...while: Similar a while pero se ejecuta al menos una vez
- for...of: Para iterar sobre elementos de arrays
- for...in: Para iterar sobre propiedades de objetos

Evita bucles infinitos
⚠️ Siempre asegúrate de que la condición de tu bucle eventualmente será falsa
⚠️ Usa break para salir temprano si es necesario
⚠️ Considera los métodos de array (map, filter, etc.) para operaciones con colecciones

Bucle for

// Imprimir números pares del 0 al 10
for (let i = 0; i <= 10; i++) {
    if (i % 2 === 0) {
        console.log(i);
    }
}

Bucle while

// Contador regresivo
let contador = 5;
while (contador > 0) {
    console.log(contador);
    contador--;
}
console.log("¡Despegue!");

Bucle do...while

// Pedir entrada hasta que sea válida
let entrada;
do {
    entrada = prompt("Introduce un número mayor que 10:");
} while (entrada <= 10 || isNaN(entrada));

console.log("Número válido:", entrada);

for...of (Arrays)

// Sumar elementos de un array
const numeros = [3, 7, 2, 9, 5];
let suma = 0;

for (const num of numeros) {
    suma += num;
}

console.log("La suma es:", suma);

for...in (Objetos)

// Listar propiedades de un objeto
const usuario = {
    nombre: "Carlos",
    edad: 28,
    email: "carlos@ejemplo.com"
};

for (const propiedad in usuario) {
    console.log(`${propiedad}: ${usuario[propiedad]}`);
}
