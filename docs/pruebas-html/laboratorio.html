🧠 Funciones Flecha y Métodos de Arrays ❓


  

    
      

        🔧 Método
        💡 Propósito
        📦 Ejemplo
      

    
    
      

        .map()
        Transforma cada elemento
        array.map(e => e + 1)
      

      

        .filter()
        Selecciona elementos que cumplen condición
        array.filter(e => e > 5)
      

      

        .reduce()
        Acumula valor a partir de elementos
        array.reduce((acc, e) => acc + e, 0)

🧭 Estructura básica de funciones funcionales ❓

  
  

    
      

        🔧 Método
        🧠 Sintaxis
        📌 ¿Qué hace?
      

    
    
      

        .map()
        array.map((e) => nuevoValor)
        Transforma cada elemento
      

      

        .filter()
        array.filter((e) => condición)
        Selecciona elementos que cumplen una condición
      

      

        .reduce()
        array.reduce((acc, e) => nuevoAcc, inicial)
        Acumula valor desde el array
      

      

        .find()
        array.find((e) => condición)
        Devuelve el primer que cumple la condición
      

      

        .some()
        array.some((e) => condición)
        Devuelve true si al menos uno cumple
      

      

        .every()
        array.every((e) => condición)
        Devuelve true si todos cumplen
      

    
  


  
💡 Esta guía permite al estudiante identificar qué método usar según su objetivo: transformar, filtrar, acumular o validar.
