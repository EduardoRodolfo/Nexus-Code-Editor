📙 Lección 1: ¿Qué es JavaScript?

JavaScript (JS) es el lenguaje que hace que las páginas web sean interactivas. HTML es estructura, CSS es estilo, JS es comportamiento.

📖 Explicación

Con JS puedes: responder clics, validar formularios, crear animaciones, conectar con servidores y mucho más.

// 📌 JS se escribe dentro de <script> // Puede ir en el <head>, al final del <body> o en archivo externo // Mensaje en consola del navegador (F12) console.log("¡Hola, mundo!"); // Mensaje emergente alert("¡Bienvenido a Nexus!"); // Mensaje en la página document.write("Hola desde JS");

🔤 Lección 2: Variables y tipos

Las variables guardan datos. En JS se usan: let, const (y antes var).

// 📦 Declarar variables let nombre = "Ana"; // Texto (string) let edad = 25; // Número (number) let activo = true; // Booleano (true/false) const PI = 3.1416; // Constante (no cambia) let datos = null; // Vacío (null) let indefinido; // undefined (sin valor) // Ver tipo de variable console.log(typeof nombre); // → "string" // Concatenar texto console.log("Hola, " + nombre + ", tienes " + edad + " años");

🧪 Ejercicio

🧮 Lección 3: Operadores

Los operadores realizan operaciones con variables.

// ➕ Aritméticos let suma = 5 + 3; // 8 let resta = 10 - 4; // 6 let mult = 3 * 4; // 12 let div = 20 / 5; // 4 let modulo = 10 % 3; // 1 (resto de 10÷3) // 🔗 Comparación (devuelven true/false) 5 == "5" // true (compara valor) 5 === "5" // false (compara valor Y tipo) 10 > 5 // true 3 <= 3 // true // ✅ Lógicos let verdadero = true && true; // AND → true let falso = true || false; // OR → true let negado = !true; // NOT → false

📝 Lección 4: Strings (cadenas)

Los strings son texto. JS tiene métodos útiles para manipularlos.

let texto = " Aprendiendo JavaScript "; // Métodos comunes console.log(texto.length); // longitud → 28 console.log(texto.toUpperCase()); // MAYÚSCULAS console.log(texto.toLowerCase()); // minúsculas console.log(texto.trim()); // quita espacios console.log(texto.includes("Java")); // ¿contiene? → true console.log(texto.replace("Java", "Type")); // reemplazar // Template strings (backticks ``) let nombre = "Ana"; console.log(`Hola, ${nombre}!`); // "Hola, Ana!"

🔢 Lección 5: Números y Math

JS tiene un objeto Math para operaciones matemáticas.

console.log(Math.PI); // 3.14159... console.log(Math.round(4.7)); // redondear → 5 console.log(Math.ceil(4.1)); // redondear arriba → 5 console.log(Math.floor(4.9)); // redondear abajo → 4 console.log(Math.random()); // aleatorio 0-1 console.log(Math.max(1, 5, 3)); // máximo → 5 console.log(Math.min(1, 5, 3)); // mínimo → 1 // Número aleatorio entre 1 y 10 let aleatorio = Math.floor(Math.random() * 10) + 1;

❓ Lección 6: Condicionales

Toma decisiones con if, else if, else.

let edad = 18; if (edad >= 18) { console.log("Eres mayor de edad"); } else { console.log("Eres menor de edad"); } // Múltiples condiciones let nota = 85; if (nota >= 90) { console.log("Excelente"); } else if (nota >= 70) { console.log("Aprobado"); } else { console.log("Reprobado"); } // Operador ternario (if simplificado) let mensaje = edad >= 18 ? "Adulto" : "Menor";

🔄 Lección 7: Bucles

Repite código varias veces con for y while.

// For: cuando sabes cuántas veces repetir for (let i = 0; i < 5; i++) { console.log(`Vuelta número ${i+1}`); } // → Vuelta número 1, 2, 3, 4, 5 // While: cuando no sabes cuántas veces let contador = 0; while (contador < 3) { console.log(`Contador: ${contador}`); contador++; } // Recorrer un array let frutas = ["manzana", "pera", "uva"]; for (let fruta of frutas) { console.log(fruta); }

📋 Lección 8: Arrays

Los arrays guardan listas de datos.

// Crear array let colores = ["rojo", "verde", "azul"]; // Acceder console.log(colores[0]); // "rojo" console.log(colores.length); // 3 // Métodos útiles colores.push("amarillo"); // agregar al final colores.pop(); // quitar el último colores.unshift("negro"); // agregar al inicio colores.shift(); // quitar el primero colores.indexOf("verde"); // buscar posición colores.includes("rojo"); // ¿está? → true/false

🎯 Lección 9: Funciones

Las funciones encapsulan código reutilizable.

// Declarar función function saludar(nombre) { return `¡Hola, ${nombre}!`; } // Usar función let mensaje = saludar("Ana"); console.log(mensaje); // "¡Hola, Ana!" // Función con varios parámetros function sumar(a, b) { return a + b; } console.log(sumar(5, 3)); // 8

🧩 Lección 10: Objetos

Los objetos agrupan datos relacionados.

// Crear objeto let persona = { nombre: "Ana", edad: 25, ciudad: "La Habana", saludar: function() { return `Hola, soy ${this.nombre}`; } }; // Acceder a propiedades console.log(persona.nombre); // "Ana" console.log(persona["edad"]); // 25 console.log(persona.saludar()); // "Hola, soy Ana" // Agregar propiedad persona.pais = "Cuba";

🎭 Lección 11: DOM

El DOM permite manipular HTML desde JS.

// Seleccionar elementos document.getElementById("miId"); document.querySelector(".clase"); document.querySelectorAll("p"); // Cambiar contenido document.getElementById("titulo").textContent = "Nuevo título"; // Cambiar estilos document.getElementById("caja").style.color = "red"; // Crear elemento let nuevo = document.createElement("p"); nuevo.textContent = "Elemento creado con JS"; document.body.appendChild(nuevo);

⚡ Lección 12: Eventos

Los eventos responden a acciones del usuario: clic, teclado, mouse...

// Evento click document.getElementById("btn").addEventListener("click", function() { alert("¡Clic!"); }); // Evento mouseover (pasar mouse) document.getElementById("caja").addEventListener("mouseover", function() { this.style.background = "#22c55e"; }); // Evento keydown (tecla presionada) document.addEventListener("keydown", function(e) { console.log(`Tecla presionada: ${e.key}`); });
Aquí se mostrarán los eventos...

🔁 Lección 13: Arrow Functions

Las arrow functions () => {} son una forma más corta de escribir funciones.

// Función normal function sumar(a, b) { return a + b; } // Arrow function (equivalente) const sumarArrow = (a, b) => a + b; // Si solo tiene un parámetro, se pueden omitir los paréntesis const cuadrado = x => x * x; // Si tiene varias líneas, usan { } y return const saludar = (nombre) => { let mensaje = `¡Hola, ${nombre}!`; return mensaje; }; console.log(sumarArrow(5, 3)); // 8 console.log(cuadrado(4)); // 16

🧪 Lección 14: Try/Catch (Manejo de errores)

Captura errores para que tu programa no se rompa.

try { // Código que puede fallar let resultado = 10 / 0; console.log("Todo bien"); // Lanzar error intencional if (resultado === Infinity) { throw new Error("División infinita"); } } catch (error) { console.error("❌ Error:", error.message); } finally { console.log("✅ Esto se ejecuta siempre"); }
