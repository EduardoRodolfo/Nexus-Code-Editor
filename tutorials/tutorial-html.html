🏗️ Lección 1: Estructura básica de HTML

HTML significa HyperText Markup Language. Es el lenguaje base de toda página web. Piensa en HTML como el esqueleto de una página.

📖 Explicación

Todo documento HTML tiene una estructura fija. Debe comenzar con <!DOCTYPE html> y contener las etiquetas <html>, <head> y <body>.

<!-- 📄 Esta es la estructura mínima de HTML --> <!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <title>Mi primera página</title> </head> <body> <h1>¡Hola, Mundo!</h1> <p>Esta es mi primera página web.</p> </body> </html>

  • Un título de pestaña: "Mi página web"
  • Un encabezado H1 que diga: "Bienvenido a Nexus"
  • Un párrafo que diga: "Estoy aprendiendo HTML"

📝 Lección 2: Encabezados (H1 a H6)

Los encabezados se usan para titular secciones. Van de <h1> (más importante) a <h6> (menos importante).

<!-- 🔤 Los 6 niveles de encabezado --> <h1>Título principal (solo uno por página)</h1> <h2>Subtítulo</h2> <h3>Sección</h3> <h4>Subsección</h4> <h5>Detalle</h5> <h6>Nota pequeña</h6>

  • H2: "Ingredientes"
  • H3: "Para la masa"
  • H3: "Para el relleno"
  • H2: "Preparación"

📄 Lección 3: Párrafos y texto

El texto en HTML se escribe dentro de etiquetas como <p> (párrafo).

<!-- 📝 Etiquetas de texto más usadas --> <p>Esto es un párrafo normal.</p> <p>Este es texto en negrita y este es <em>texto en cursiva</em>.</p> <p>También podemos <u>subrayar</u> texto.</p> <!-- Salto de línea --> <p>Primera línea<br>Segunda línea</p>

🧪 Ejercicio

✏️ Crea un párrafo con formato:

Escribe un párrafo que hable sobre ti usando negrita, cursiva y saltos de línea.

🔗 Lección 4: Enlaces

Los enlaces se crean con <a> y permiten navegar entre páginas.

<!-- 🌐 Tipos de enlaces --> <!-- Enlace externo --> <a href="https://ejemplo.com">Ir a ejemplo</a> <!-- Enlace interno (misma página) --> <a href="#seccion2">Ir a sección 2</a> <!-- Enlace que abre en nueva pestaña --> <a href="https://ejemplo.com" target="_blank">Abrir en nueva pestaña</a>

🖼️ Lección 5: Imágenes

Las imágenes se insertan con <img>. No necesita etiqueta de cierre.

<!-- 🖼️ Insertar imagen --> <img src="foto.jpg" alt="Descripción de la imagen"> <!-- Con ancho y alto --> <img src="foto.jpg" width="300" height="200" alt="Descripción">

📋 Lección 6: Listas

Las listas pueden ser ordenadas (<ol>) o desordenadas (<ul>).

<!-- 📋 Lista desordenada (viñetas) --> <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> <!-- Lista ordenada (números) --> <ol> <li>Primer paso</li> <li>Segundo paso</li> </ol>

📊 Lección 7: Tablas

Las tablas organizan datos en filas y columnas.

<!-- 📊 Tabla simple --> <table> <tr> <th>Nombre</th> <th>Edad</th> </tr> <tr> <td>Ana</td> <td>25</td> </tr> </table>

🧱 Lección 8: Divisiones (div)

<div> agrupa contenido para darle estilo o estructura.

<!-- 🧱 Divisiones para estructurar --> <div class="header"> <h1>Mi sitio web</h1> </div> <div class="contenido"> <p>Aquí va el contenido principal.</p> </div> <div class="footer"> <p>© 2026 Mi sitio</p> </div>

📝 Lección 9: Formularios

Los formularios recogen datos del usuario.

<!-- 📝 Formulario básico --> <form action="/guardar" method="POST"> <label>Nombre:</label> <input type="text" name="nombre"> <label>Email:</label> <input type="email" name="email"> <button type="submit">Enviar</button> </form>

⌨️ Lección 10: Tipos de Input

Hay muchos tipos de input para diferentes datos.

<!-- ⌨️ Tipos de input más usados --> <input type="text"> <!-- Texto --> <input type="password"> <!-- Contraseña --> <input type="email"> <!-- Email --> <input type="number"> <!-- Número --> <input type="date"> <!-- Fecha --> <input type="color"> <!-- Color --> <input type="file"> <!-- Archivo --> <input type="checkbox"> <!-- Casilla --> <input type="radio"> <!-- Botón de opción -->

🎨 Lección 11: Estilos en línea

Se puede agregar CSS directamente a una etiqueta con el atributo style.

<!-- 🎨 Estilos en línea --> <p style="color: blue; font-size: 20px;"> Este texto es azul y grande. </p> <div style="background: #f0f0f0; padding: 20px; border-radius: 8px;"> <h2>Sección con estilo</h2> </div>

🎵 Lección 12: Audio y Video

HTML5 permite insertar multimedia directamente.

<!-- 🎵 Audio --> <audio controls> <source src="musica.mp3"> Tu navegador no soporta audio. </audio> <!-- 🎬 Video --> <video width="320" height="240" controls> <source src="video.mp4"> Tu navegador no soporta video. </video>

📐 Lección 13: Semántica HTML5

Las etiquetas semánticas dan significado al contenido.

<!-- 📐 Etiquetas semánticas --> <header>Encabezado de la página</header> <nav>Menú de navegación</nav> <main>Contenido principal</main> <section>Una sección</section> <article>Un artículo</article> <aside>Contenido lateral</aside> </main> <footer>Pie de página</footer>

♿ Lección 14: Accesibilidad

Haz tus páginas accesibles para todos.

<!-- ♿ Buenas prácticas de accesibilidad --> <!-- Texto alternativo en imágenes --> <img src="logo.png" alt="Logo de Nexus Code Editor"> <!-- Etiquetas en formularios --> <label for="nombre">Nombre completo:</label> <input type="text" id="nombre" name="nombre"> <!-- Roles ARIA --> <button aria-label="Cerrar ventana">X</button>
