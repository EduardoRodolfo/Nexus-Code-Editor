🎨 Lección 1: ¿Qué es CSS?

CSS (Cascading Style Sheets) da estilo a la web. HTML es el esqueleto, CSS es la ropa.

📖 Explicación

Se usa de 3 formas:

/* 1️⃣ En línea */ <p style="color:red">Texto</p> /* 2️⃣ En el <head> */ <style> p { color: blue; } </style> /* 3️⃣ Archivo externo ✅ */ <link rel="stylesheet" href="estilos.css">

🧪 Ejercicio

Crea un HTML con CSS externo: título verde, fondo gris claro.

✏️ Lección 2: Sintaxis básica

Regla CSS: selector { propiedad: valor; }

h1 { color: #00d4aa; font-size: 32px; text-align: center; } p { color: #e2e8f0; line-height: 1.6; }

🧪 Ejercicio

CSS para que todos los párrafos tengan color azul, 18px, centrados.

🔗 Lección 3: Colores y fondos

Colores: nombre, hex (#rrggbb), rgb(), degradados.

/* Nombres */ p { color: red; } /* Hexadecimal */ h1 { color: #00d4aa; } /* RGB */ div { color: rgb(100,200,150); } /* Degradado */ .caja { background: linear-gradient(135deg, #0a1628, #0d2137); }
🎨 Degradado en vivo

🧪 Ejercicio

Div con fondo degradado azul a morado y texto blanco.

📝 Lección 4: Tipografía

Controla fuentes, tamaños, pesos y estilos de texto.

body { font-family: 'Segoe UI', Arial, sans-serif; font-size: 16px; font-weight: 400; line-height: 1.8; } h1 { font-size: 2.5em; font-weight: 700; } .italic { font-style: italic; } .uppercase { text-transform: uppercase; }
📝 Georgia, 1.2em - Texto de ejemplo

📦 Lección 5: Modelo de caja

Todo elemento HTML es una caja con: content, padding, border, margin.

/* Cada elemento ocupa espacio así: */ div { width: 200px; padding: 20px; border: 2px solid #22c55e; margin: 10px; /* Total: 200 + 20+20 + 2+2 + 10+10 = 264px */ } /* box-sizing: border-box incluye padding y border en width */ * { box-sizing: border-box; }

📐 Lección 6: Márgenes y paddings

Margin = espacio fuera del elemento. Padding = espacio dentro.

/* Márgenes */ div { margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px; /* Shorthand: margin: 10px 20px; */ } /* Paddings */ div { padding: 15px 30px; /* 15px arriba/abajo, 30px izq/der */ }

🖼️ Lección 7: Bordes y sombras

Bordes decorativos y sombras para dar profundidad.

/* Bordes */ .caja { border: 2px solid #22c55e; border-radius: 12px; } /* Sombras */ .caja { box-shadow: 0 8px 30px rgba(0,0,0,.4); } /* Sombra de texto */ h1 { text-shadow: 2px 2px 4px rgba(0,0,0,.5); }
🖼️ Borde verde + esquinas redondeadas
✨ Sombra suave

🧭 Lección 8: Flexbox

Flexbox distribuye elementos en una fila o columna sin esfuerzo.

.contenedor { display: flex; gap: 10px; justify-content: center; align-items: center; flex-wrap: wrap; } .item { flex: 1; text-align: center; }

✨ Lección 10: Animaciones

CSS puede animar elementos sin JavaScript.

@keyframes mover { from { transform: translateX(0); } to { transform: translateX(200px); } } .caja { animation: mover 2s infinite alternate; }

📱 Lección 11: Responsive Design

Haz que tu web se vea bien en cualquier pantalla.

/* Media queries */ @media (max-width: 600px) { body { font-size: 14px; } .columna { flex-direction: column; } }
📱 Cambia el tamaño de la ventana (≤400px cambia a morado)

🎭 Lección 12: Pseudoclases

Estados especiales de los elementos: :hover, :focus, :nth-child.

/* Pasa el mouse */ a:hover { color: #22c55e; background: #2a3a5c; } /* Elemento en posición par */ li:nth-child(even) { background: #1a1a2e; }
🖱️ Pasa el mouse aquí

🧩 Lección 13: Variables CSS

Guarda valores reutilizables en variables (custom properties).

/* Definir variables */ :root { --color-primario: #00d4aa; --color-fondo: #0a0e1a; --radius: 8px; } /* Usar variables */ button { background: var(--color-primario); border-radius: var(--radius); }

🔧 Lección 14: Selectores avanzados

Selecciona elementos específicos sin clases extras.

/* Hijo directo */ div > p { color: #22c55e; } /* Hermano adyacente */ h2 + p { font-weight: 700; } /* Atributo */ input[type="email"] { border-color: #6366f1; } /* Negación */ div:not(.excluido) { opacity: 1; }
