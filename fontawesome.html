📖 Cómo usar Font Awesome en tus proyectos

Agrega esta línea en el <head> de tu HTML para usar Font Awesome desde tu servidor local:

📄 HTML 
<link rel="stylesheet" href="fontawesome-free-7.2.0-web/css/all.min.css">

También puedes combinar con Bootstrap local (desde node_modules/):

📄 HTML 
<link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.min.css">

🔷 Font Awesome Solid (fas)

Clase: fas fa-nombre — Los iconos sólidos son los más usados y versátiles.

🔷 Font Awesome Regular (far)

Clase: far fa-nombre — Iconos con estilo de contorno, más sutiles.

🔷 Font Awesome Brands (fab)

Clase: fab fa-nombre — Logos de marcas, redes sociales y tecnologías.

📏 Tamaños de Iconos

Font Awesome usa clases fa-xs, fa-sm, fa-lg, fa-2x hasta fa-10x

Clase Ejemplo Código
fa-xs <i class="fas fa-star fa-xs"></i>
fa-sm <i class="fas fa-star fa-sm"></i>
fa-lg <i class="fas fa-star fa-lg"></i>
fa-2x <i class="fas fa-star fa-2x"></i>
fa-3x <i class="fas fa-star fa-3x"></i>
fa-5x <i class="fas fa-star fa-5x"></i>
fa-10x <i class="fas fa-star fa-10x"></i>

🎨 Colores

Los iconos heredan el color del texto. Usa clases de Bootstrap o estilo style="color: ...".

style="color: #ef4444;"

style="color: #22c55e;"

style="color: #528dd7;"

style="color: #f59e0b;"

📄 Código 
<i class="fas fa-heart fa-3x" style="color: #ef4444;"></i>

🔄 Rotación y Flip

Clases: fa-rotate-* (90, 180, 270) y fa-flip-* (horizontal, vertical, both)

Normal
fa-rotate-90
fa-rotate-180
fa-rotate-270
fa-flip-horizontal
fa-flip-vertical
fa-flip-both

</button>
