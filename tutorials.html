📖 Tutorial 1: Comandos Básicos de Linux

⏱️ 15 minutos 🎯 Principiante ✅ 0/5 ejercicios

Aprende los comandos esenciales para navegar y manipular archivos en Linux. Perfecto para quienes empiezan con sistemas Unix.

✏️ Tutorial 2: Editores de Texto y Scripts

⏱️ 20 minutos 🎯 Principiante 🔒 Requiere Tutorial 1

Domina editores como nano y vim, y crea tus primeros scripts en Bash para automatizar tareas.

⚙️ Tutorial 3: Compilación con GCC

⏱️ 25 minutos 🎯 Intermedio 🔒 Requiere Tutorial 2

Usa el compilador GCC incluido en JS Linux para compilar programas en C y entender el proceso de compilación.

⏱️ 30 minutos 🎯 Avanzado 🔒 Requiere Tutorial 3

Crea scripts complejos con variables, condicionales, bucles y funciones en Bash.
