¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que da estilo y diseño a las páginas web. Controla colores, fuentes, márgenes, disposición y animaciones. Funciona junto con HTML.

🧪 Sin CSS → texto plano. Con CSS → esto es azul y con estilo.

HTML: estructura CSS: presentación

Se escribe en reglas compuestas por selector { propiedad: valor; }.

Sintaxis básica

Una regla CSS tiene tres partes: selector, propiedad y valor.

selector { propiedad: valor; otra-propiedad: otro-valor; }

Ejemplo real:

h1 { color: #2c3e50; font-size: 2rem; text-align: center; }
💡 Puedes escribir CSS dentro de una etiqueta <style> o en un archivo .css externo.

Selectores

Los selectores apuntan a los elementos HTML que queremos estilizar.

🔹 Selector de elemento

p { color: gray; }

🔸 Selector de clase (.)

.destacado { background: yellow; font-weight: bold; }
Texto con clase "destacado" (simulado)

<p class="destacado">Hola</p>

🔹 Selector de ID (#)

#header-principal { background: navy; color: white; }

📦 Box Model

Todos los elementos son cajas rectangulares. Se componen de: contenido, padding, borde y margen.

Contenido
padding · border · margin
.caja { width: 200px; padding: 20px; border: 5px solid #eab308; margin: 20px; }

Usa box-sizing: border-box; para que width incluya padding y border (más intuitivo).

🧩 Flexbox

Flexbox es un sistema unidimensional para alinear y distribuir espacio entre elementos.

Item 1
Item 2
Item 3

display: flex; gap: 0.75rem;

.contenedor { display: flex; justify-content: space-between; align-items: center; flex-wrap: wrap; }

Propiedades clave: justify-content, align-items, flex-direction.

📱 CSS Grid

Grid permite crear diseños bidimensionales (filas y columnas).

display: grid; grid-template-columns: repeat(3, 1fr);

.grid-container { display: grid; grid-template-columns: 1fr 2fr 1fr; gap: 10px; }

Ideal para layouts complejos. Combínalo con media queries.

📲 Media Queries · Responsive

Las media queries aplican estilos según el tamaño de pantalla u otras características.

@media (max-width: 768px) { .sidebar { display: none; } .contenido { width: 100%; } }
📱 Este tutorial es responsivo: redimensiona la ventana para ver cómo cambia el índice, las columnas y tamaños.

Puntos de ruptura comunes: 480px, 768px, 1024px.
